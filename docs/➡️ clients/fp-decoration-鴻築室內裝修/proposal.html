05

各頁重點

首頁 index.html

Hero 主視覺（含背景圖）快速建立品牌第一印象，依序帶出關於鴻築預覽、服務項目六格、工程實績五格、最新消息預覽，各區塊均設有導向內頁的 CTA 按鈕。

關於鴻築 about.html

頂部 Banner → 創立紀事（圖文並排，含設計情境照片）→ 發展歷程時間軸（2010 / 2015 / 現今）→ 五部門卡片（管理、業務、設計、工務、規劃）。

服務項目 services.html

頂部 Banner → 六大服務卡片 → 合作流程四步驟（需求洽談 → 規劃設計 → 施工執行 → 驗收交付）。

工程實績 portfolio.html

頂部 Banner → 五格作品展示（購物中心、酒店旅館、百貨公司、住宅公設、複合式商業）。圖片置於 images/profolios/，上線前建議依案例類型重新命名檔案。

最新消息 news.html

頂部 Banner → 最新消息三格 → 員工活動三格 → 友站連結（合作夥伴 / 產業資源 兩欄）。

06

技術與維運建議

前端架構HTML / CSS 共用 styles.css，搭配 body class（page-home / page-about 等）做頁面差異樣式，結構清晰可快速上線。
SEO各頁尚未加入 <meta name="description">，建議上線前補齊所有頁面的 meta 描述。
聯絡表單目前為靜態 HTML，建議串接 Formspree 或 Netlify Forms，實現表單送出通知。
多語系en/ 目錄入口已預備（nav 已有繁中｜EN 切換），英文版頁面開發優先順序待確認。
圖片管理portfolio 圖片置於 images/profolios/，檔名建議依案例類型整理（現有檔名皆為 hotel-*）。
後續升級CMS（利於消息與案例維護）、GA4 分析追蹤、案例詳情頁模板、文章系統。
