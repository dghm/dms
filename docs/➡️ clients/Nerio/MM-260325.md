## 三、討論重點與功能需求

### 3.1 報價項目

| 項目         | 金額         |
| ------------ | ------------ |
| 拍面設計     | $3,500       |
| Logo 設計    | $10,000      |
| 簡易網站設計 | $30,000 以內 |

### 3.2 品牌資訊

| 欄位        | 內容                                |
| ----------- | ----------------------------------- |
| 品牌名稱    | Nerio                               |
| 公司全名    | 御勢行銷有限公司 Nerio Strategy Lab |
| 子品牌      | Nerio Pop Lab                       |
| Slogan 候選 | Lead the Shift / Pick up Your Life  |
| 客戶背景    | 行銷及廣告業出身                    |
| 主要通路    | 網路商場（電商）                    |

### 3.3 品牌概念與定位

- **品牌精神**：戰士／王者形象（Nerio 語源意涵）
- **市場路線**：消費導向、中階定價，走大眾潮流路線
- **核心受眾**：大眾化、潮流文化，以女性為主，涵蓋成人與兒童市場
- **品牌感受**：引領潮流、優越感、令人眼睛為之一亮
- **關鍵詞**：Pop 流行、LAB 實驗精神、「找到最後一片拼圖」

### 3.4 產品線

- 設計毛巾
- 包包
- 球拍
- 鞋子（主打速度與平衡）

### 3.5 Logo 設計方向

**風格**

- 簡單線條＋幾何為主
- 有對比性、設計感與質感並存
- 參考：ON 昂跑的線條處理方式

**色彩**

- 2025–2026 流行咖啡色
- 女性市場：鮮艷／潮流色系、簡約配色
- 大眾女性偏好色，螢光黃可納入考量

### 3.6 網站需求

- 電商套版網站
- 平台選項：Cyberiz 或 Shopline（待確認）

---

## 四、討論重點摘要

1. 本次確認 Nerio 的品牌精神、主要受眾與產品線方向，作為視覺系統的基底。
2. Logo 採「簡單線條＋幾何」並搭配潮流配色方向（含咖啡色與螢光黃考量），以符合女性市場定位。
3. 網站以電商套版形式進行，需於 `Cyberiz` 與 `Shopline` 間完成最終選擇後再定案。
