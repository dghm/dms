第一條　定義
  1. 「本合約」：指本服務合約本文及所有附件，具同等效力。
  2. 「服務」：指乙方依本合約為甲方建置 Airtable Interface 系統及交付操作說明文件，詳如附件一至附件六所載。
  3. 「工作成果」：指乙方依本合約完成之 Airtable Interface 建置、自動化邏輯設定及操作說明文件（含所有相關設定、邏輯與文件交付物）。
  4. 「UAT 文件」：指乙方於各專案完成後提供之使用者驗收測試清單，作為各系統驗收之依據。
  5. 「驗收」：指甲方依 UAT 文件完成測試確認，並以書面通知乙方通過之程序。
  6. 「工作天」：指週一至週五（不含國定假日）之正常工作日。
第二條　服務範疇
  1. 乙方應依本合約及各附件報價單所載範疇提供服務，各附件之報價單與本合約具同等效力。
  2. 本合約服務範疇以各附件所載項目為限。未載明之功能、自動化邏輯、系統整合、維護或後續功能擴充，均不在本合約範圍內，如需另行辦理，應另行報價並以書面確認。
  3. 附件五（DH-01-PJ11-QT-1b）第三項「Taxable Charge 計價資料項目合併作業套件」為可選加購項目，須由甲方書面確認後方納入本次服務範疇；未確認者視為不採購，不影響附件五第一、二項之執行。
第三條　專案時程
  1. 起始日：工作時程自乙方開立發票並送達甲方，且甲方以書面或電子郵件確認收到後次一工作日起算。
  2. 各系統預計工作天數如下：
    系統 開發期
    CRM Interface (TW) 約 10 工作天
    OMS Interface (TW) 約 21 工作天
    FIN Interface (TW) 約 14 工作天
    上述時程為預估值。各系統之 AWB/POD 填表自動化及 Debit Note 計費自動化套件之工作時程，包含於對應系統（OMS / FIN）之開發期內。
  3. 整合測試期：三系統開發完成後，進行跨系統整合測試，預計約 10 工作天，以確保系統運作正確、資料登載無誤及各系統間資料流通順暢。
  4. 操作說明文件撰寫期：整合測試完成後進行操作說明文件撰寫（附件六），預計約 21 工作天。
  5. 整體工作流程順序為：CRM / OMS / FIN 系統開發 → 整合測試 → 操作說明文件撰寫 → 驗收。各系統開發期得視資源排程同步進行。
  6. 若甲方未於約定期限內提供乙方執行所需之資料、帳號、系統存取權限或回饋意見，對應工作時程得依延遲天數順延，且不視為乙方違約。

第五條　驗收程序
  1. 各系統建置完成後，乙方應提供 UAT 文件予甲方，由甲方進行使用者驗收測試。
  2. 甲方應於收到 UAT 文件後一（1）個月內完成測試，並以書面通知乙方測試結果。
  3. 如測試期間發現不符 UAT 文件所載事項之問題，甲方應具體說明，乙方應於合理期限內進行修正，修正完成後甲方應重新進行相關項目之測試。
  4. 甲方逾期未回覆，視為驗收通過。
  5. 驗收範圍以 UAT 文件所載事項為準，不包含原規格外之新增需求或功能調整。
  6. 操作說明文件（附件六）之驗收：甲方於收到文件後，包含一次完整校稿回饋，乙方進行修正後即視為交付完成。
第六條　範疇變更
  1. 任何超出各附件所載範疇之需求，雙方均須以書面提出變更申請（Change Request）。
  2. 乙方收到變更申請後，應提供變更評估，包含費用影響及時程調整建議。
  3. 雙方書面確認後，變更方正式生效並納入本合約範疇。口頭承諾不具拘束力。
第七條　智慧財產權
  1. 本合約所建置之工作成果，其著作權及相關智慧財產權歸乙方（萬能數維有限公司）所有。
  2. 甲方於本合約服務費用全額結清後，就本合約工作成果取得永久、不可撤銷之使用授權，得於其內部業務需求範圍內使用。
  3. 甲方在 Airtable 系統中所填寫、儲存之業務資料（包含但不限於客戶資料、訂單資料、財務資料），屬甲方所有，乙方不得將其用於本合約目的以外之用途。
  4. 操作說明文件（附件六）交付後，甲方得於其內部使用，不得轉售、轉授權或對外發佈。
第八條　保固
  1. 各系統驗收通過後，乙方提供三（3）個月保固服務。
  2. 保固期自各系統個別驗收通過日起算。
  3. 保固範圍限於原建置功能之異常修正，不包含以下情況：
    1. 新增功能或規格調整
    2. 因甲方操作不當或資料錯誤所引發之問題
    3. 因 Airtable 平台政策異動、版本更新或服務中斷所造成之功能失效
    4. 附件五第三項可選套件中，因 Airtable Script 平台限制或 API 政策異動導致之功能異常
  4. 保固期屆滿後如需持續維護支援，將另行報價。
第九條　第三方平台
  1. 本合約服務建置於 Airtable 第三方平台之上。Airtable 之帳號、權限與訂閱費用由乙方代為訂閱與管理，實際費用依第三方服務當時之官方定價調整，乙方不保證其價格、政策或服務之穩定性不變。
  2. 乙方對 Airtable 平台之政策變更、定價調整、功能異動或服務中斷，不負任何賠償責任。
