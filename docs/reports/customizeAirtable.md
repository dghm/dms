## 二、DGHM 資料庫服務的核心定位

DGHM 的「資料庫客製化服務」並非傳統商業軟體意義上的客製化，而是以**學術驅動型資料庫建置模式**呈現，主要涵蓋以下三大方向：

### 🔬 1. 微生物基因組學資料庫
- 採用 **Abricate** 及 **NCBI AMRFinderPlus** 資料庫為基礎，建立客製化內部（In-house）分析流程（Pipeline）
- 針對整個基因組定序（Whole Genome Sequencing, WGS）資料進行抗藥性基因分析
- 在基因型（Genotype）與表現型（Phenotype）不一致時，提供重新分析機制與額外網路資料庫比對服務

### 🍄 2. FunResDB——真菌抗藥性客製化資料庫

FunResDB 是 DGHM 最具代表性的資料庫客製化成果之一：

| 項目 | 說明 |
|------|------|
| **資料庫名稱** | FunResDB（真菌抗藥性資料庫） |
| **目標病原** | 煙麴黴菌（*Aspergillus fumigatus*） |
| **核心功能** | 收集並儲存所有關於 CYP51A 依賴性抗藥性的完整資訊 |
| **客製化工具** | 提供量身打造（Tailor-made）的線上序列分析工具，支援基因型藥物敏感性測試 |
| **資料涵蓋** | 79 種 CYP51A 變體，連結至 59 個非同義突變（Nonsynonymous Mutations） |
| **服務形式** | 整合性網路服務平台（Curated Web-based Service） |
| **主要用途** | 研究並監測以 cyp51a 序列為基礎的真菌藥物敏感性 |

> 📌 **客製化亮點：** FunResDB 整合了資料庫與分析工具，使研究人員能針對不同菌株進行個別化（個案客製化）的基因型敏感性測試，是 DGHM 資料庫客製化服務的最佳範例。

### 🦠 3. ARVIA——抗生素抗藥性監測資料庫

| 項目 | 說明 |
|------|------|
| **資料庫名稱** | ARVIA（Antibiotic Resistance Visualization and Analysis） |
| **資料來源** | ARS（德國抗生素抗藥性監測系統）資料庫 |
| **ARS 定位** | 以實驗室為基礎的志願性監測系統，持續收集醫院與門診的抗藥性數據 |
| **客製化功能** | 評估 ARVIA 是否能偵測抗生素消費量與抗藥性之間的關聯性 |
| **數據涵蓋** | 涵蓋所有臨床病原體與樣本類型的完整抗藥性統計 |
| **服務對象** | 德國公衛機關、醫院、門診醫療機構 |

## 三、客製化資料庫的技術特色

根據 DGHM 歷年學術會議（包括 2025 年耶拿年會）的摘要與論文，其資料庫客製化服務具備以下技術特色：

### 🧬 3.1 基因組監測與客製化流程
- **GENTRAIN 計畫：** 開發數位化基因組感染鏈分析工具，供德國公衛機關使用，範圍延伸至 SARS-CoV-2 以外的其他重要病原體
- **內建流程（In-house Pipeline）：** 基於 Abricate 與 NCBI AMRFinderPlus 建立的客製化 WGS 分析流程，針對不同機構需求進行調整
- **基因組監測：** 如腸球菌（*Enterococcus faecium*）、產碳青黴烯酶腸道菌（CPE）的基因組監測與抗藥性分析

### 📊 3.2 資料整合與視覺化
- 結合多種公開資料庫（NCBI、ARS 等）進行交叉比對
- 提供客製化板型格式（Customized Plate Formats）
- 以視覺化儀表板形式呈現監測結果，支援動態更新

### 🔗 3.3 網路化整合服務
- 所有客製化資料庫均以**網路服務平台**形式提供
- 整合分析工具與資料庫於同一平台，降低使用門檻
- 支援序列上傳、突變比對、敏感性報告一站式輸出

## 五、DGHM 資料庫服務的品質標準

值得一提的是，DGHM 所制定的**微生物品質標準（DGHM 11.1 2018）**亦被廣泛應用於食品產業的品質管控：

- 要求產品不含**沙門氏菌（Salmonella）**、**大腸桿菌（E. coli）**及其他微生物污染物
- 此標準已被 IFS 認證機構（國際食品標準認證）採納為品質驗證依據
- 顯示 DGHM 的資料庫與標準制定能力已從醫學領域延伸至食品安全領域

## 六、DGHM 最新發展（2025 年）

根據 2025 年 DGHM 耶拿年會（DGHM 2025 Jena）的資料，DGHM 持續擴展以下資料庫客製化相關工作：

1. **腸球菌基因組監測資料庫：** 德國全國參考中心（NRC）於 2024 年完成 *Enterococcus faecium* 基因組監測與抗藥性分析
2. **CPE 整合基因組監測：** 針對 *E. coli* 及 *Klebsiella pneumoniae* 的碳青黴烯酶流行病學特徵分析
3. **肺部器官晶片模型資料庫：** 建立侵襲性毛黴菌病（Mucormycosis）的感染動態資料庫
4. **次世代定序（NGS）診斷整合：** 以 cfDNA（無細胞 DNA）測序診斷嚴重真菌感染，並整合至現有資料庫系統
