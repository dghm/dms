# DGHM Document Management System

Welcome to the DGHM Document Management System!

This is the DGHM Document Management Center, containing all related technical documentation, API documentation, user guides, and more.

## Getting Started

### Development Environment Requirements

- [Node.js](https://nodejs.org/en/download/) version 20.0 or above:
  - When installing Node.js, it is recommended to check all dependency-related options.

### Starting the Development Server

Execute the following commands in the `dms` directory:

```bash
npm install
npm start
```

The development server will start at http://localhost:3000/ and will automatically reload your changes.

### Building for Production

```bash
npm run build
```

After the build is complete, you can preview it using:

```bash
npm run serve
```
