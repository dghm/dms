# DGHM 文件管理系統

歡迎來到 **DGHM 文件管理系統（DMS）**！

這是一個集中式的文件管理平台，用於存放和管理 DGHM 專案的所有相關技術文檔、操作手冊、API 說明和使用指南。

## 📚 系統簡介

DGHM 文件管理系統提供了一個統一的平台，讓團隊成員可以輕鬆查找、閱讀和管理專案相關的文件。系統採用 [Docusaurus](https://docusaurus.io/) 建置，提供現代化的閱讀體驗和強大的搜尋功能。

## 📁 文檔結構

本系統主要包含以下文檔類別：

### TailorMed 專案文檔

#### Data（資料庫）

- **CRM**：客戶關係管理系統相關文檔
  - 系統介紹、資料結構、自動化流程
- **FIN**：財務系統相關文檔
  - 系統介紹、資料結構、自動化流程、工作流程圖
- **OMS**：訂單管理系統相關文檔

#### Interface（介面）

- **CRM**：CRM 系統介面相關文檔
  - 系統介紹
  - Role and Context：角色與上下文
  - Data Logic：資料邏輯
  - Interface Design：介面設計

#### Website（網站）

- **2025 版本**：2025 年網站相關文檔
  - 首頁、關於我們、解決方案、專業領域、最新消息、聯絡我們
- **2026 版本**：2026 年網站改版相關文檔
  - 網站改版工作流程說明（WF 說明）
  - 參考資料和文檔模板
  - 文案內容

### 操作手冊

提供系統使用、文檔編輯、部署等相關的操作指南：

- **文檔編輯指南**：如何編輯和新增文檔
- **Markdown 快速參考**：Markdown 語法快速查詢
- **Netlify 部署指南**：網站部署相關說明
- **標籤功能說明**：如何使用標籤功能
- **雙語設定說明**：多語言文檔設定指南
- **其他操作說明**：各種功能的使用指南

## 🔍 快速導航

### 尋找特定文檔

- 使用**搜尋功能**（右上角搜尋框）快速找到需要的文檔
- 瀏覽**側邊欄**按照分類查找文檔
- 使用**標籤**功能篩選相關文檔

## 🎯 使用說明

### 權限說明

根據您的帳號權限，您可能只能看到部分文檔：

- **Admin 帳號**：可以查看所有文檔
- **TailorMed 帳號**：只能查看 TailorMed/Website/2026/網站改版/WF 說明 目錄下的內容

### 瀏覽文檔

1. 使用左側**側邊欄**瀏覽文檔分類
2. 點擊文檔標題即可閱讀內容
3. 使用**搜尋功能**快速找到特定主題

### 編輯文檔（限有權限的用戶）

1. 參考[文檔編輯指南](/docs/操作手冊/文檔編輯指南)
2. 使用 Markdown 語法撰寫內容
3. 參考[Markdown 快速參考](/docs/操作手冊/Markdown快速參考)查詢語法

## 💡 開始使用

選擇左側側邊欄中的文檔分類，開始探索您需要的文檔內容。

如有任何問題或需要協助，請參考[操作手冊](/docs/操作手冊/README_客戶使用指南)或聯絡技術支援。
